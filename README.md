Lerna 7+ & @rnx-kit/tools-workspaces bug example

### Requirements
* globally available `pnpm`

### Fetch
```
> git clone git@github.com:poniraq/lerna7-tools-workspaces-bug-example.git
```

### Setup
```
> cd lerna7-tools-workspaces-bug-example
> pnpm install
```

### Run
```
> pnpm start
```
